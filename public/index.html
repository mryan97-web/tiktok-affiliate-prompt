📐 Urutan Penyusunan
  1. System Rules — aturan dasar AI (wajib)
  2. Character DNA — identitas karakter (wajib, sebelum scene apapun)
  3. Identity Lock — proteksi perubahan karakter (wajib, sebelum pose/produk)
  4. Scene Module — latar tempat (wajib, fase 2) FASE 2
  5. Product Module — objek produk (wajib, fase 2) FASE 2
  6. Pose Module — gestur karakter (opsional, fase 2) FASE 2
  7. Camera Module — sudut kamera (opsional, fase 3) FASE 3
  8. Lighting Module — pencahayaan (opsional, fase 3) FASE 3
  9. Branding Module — brand overlay (opsional, fase 4) FASE 4
  10. Negative Prompt — larangan AI (wajib, akhir)
🔗 Dependency Graph
Character DNA ← wajib sebelum Scene, Product, Pose
Identity Lock ← wajib diterapkan sebelum Pose & Product
Scene Module ← butuh Character DNA untuk konteks
Product Module ← butuh Pose + Identity Lock
Prompt Assembly ← tahap akhir, menggabungkan semua modul
